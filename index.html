¿Que es JS?

Es un lenguaje multiproposito util para escribir y programar codigo. Es usado tanto en fronend como backend. Brinda el dinamismo, versatilidad y desenvolvimiento a los sitios webs

Admite diversos frameworks como:

  • Angular
  • React
  • Vue

Tipos de datos

Para las computadoras, los datos son cualquier tipo de informacion que procesen

Javascript admite 8 tipos de datos

  • Strings
  • Arrays
  • Numbers
  • Objects
  • Boolean
  • Symbol
  • Null
  • Undefined

Los strings o cadenas, se identifican con '' o "". Añadiendoles + entre las '' se pueden formar concatenaciones mas complejas

los arrays estan dentro de []. Se puede acceder a ellos y modificarlos a traves de la variable y su numero indice

var arr1 = [1,2,3]; arr1.push(4); // arr1 is now [1,2,3,4]
Operadores

son aquellos que permiten realizar una operacion matematica o representan algo en especifico

  • =, lo que esta a la derecha es el valor del lado izquierdo
  • ==, igual que
  • ===, estrictamente igual
if/else

If indica que se ejecute el codigo dentro de la declaracion, de ser verdadero se ejecutara. else indicara que codigo ejecutar en caso de que el resultado dentro de la declaracion sea falso

if (num > 10) { return "Bigger than 10"; } else { return "10 or Less"; }

en ocasiones que hay multiples opciones se utilisa else if

if (num > 15) { return "Bigger than 15"; } else if (num < 5) { return "Smaller than 5"; } else { return "Between 5 and 15"; }
Objetos

son similares a los arrays, contenedores de datos, en este caso contenedores de datos que asemejan objetos en la vida real, describe las caracteristicas del objeto.Al objeto lo conforman las propiedades, este se puede modificar a traves de ellas. las propiedades bien pueden ser strings o numeros. en caso de olvidar algun detalle de string, JS lo autocompletara.

var cat = { "name": "Whiskers", "legs": 4, "tails": 1, "enemies": ["Water", "Dogs"] };
Loops

una secuencia de codigo se puede ejecutar multiples veces con loop. Se usan las declaraciones while loop, for loop, do while y recursion

var ourArray = []; var i = 0; while(i < 5) { ourArray.push(i); i++; }
Calculos matematicos

En JS se pueden hacer las operaciones matematicas basicas: +,-, / y *.
